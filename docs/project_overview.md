# Project Overview

## What INANNA NYX Is

INANNA NYX is a constitutional intelligence architecture for readable digital bodies, communal stewardship, and future civic-scale machine dialogue.

It is not centered on one runtime, one model, one operating system, or one product form. It is centered on a deeper structural question:

How can intelligence, system bodies, governance, memory, proposal, and communal life exist together without collapsing into opacity, domination, or fragmentation?

## Why This Repository Exists

This repository exists because the full realization of INANNA NYX exceeds current resources, current team size, and the practical power of currently available coding agents.

That truth does not reduce the value of the project. It changes the phase.

The present phase is one of **architectural seed formation**:

- preserve the spine
- define the law
- clarify the beings
- describe the governance paradigm
- make the guardianship layer explicit
- protect the civic horizon
- prepare a clean handoff for future builders

## Why The Reset Was Necessary

Earlier project spaces mixed together:

- constitutional vision
- practical MVP thinking
- implementation experiments
- symbolic and narrative layers

That mixture made inheritance harder. This repository starts from zero so the project can be read clearly.

## What Is Primary Here

The primary center of this repository is:

- constitutional clarity
- explicit law
- readable ontology
- governance above the model
- plurality without fragmentation
- truthful inheritance

## What Is Secondary Here

The older practical MVP remains meaningful as a candidate first manifestation, especially because it preserves a credible low-resource path. But it is secondary here. It is not the constitutional center.

## What Future Builders Should Take From This Repo

A future builder should be able to answer:

- what this project is for
- what must never be betrayed
- what entities and layers exist in its universe
- what kind of governance it seeks
- what kind of civic life it serves
- what a believable first proof could be
- what remains intentionally deferred
